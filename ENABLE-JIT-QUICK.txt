📌 MÉTODO 1: MANUAL (5 minutos)
═══════════════════════════════════════════════════════════════════

1️⃣  Abrir php.ini:
   → Laragon (ícone na bandeja)
   → Botão direito > Menu > PHP > php.ini
   
   OU usar o caminho:
   C:\laragon\bin\php\php-8.2.29\php.ini

2️⃣  Encontrar seção [opcache]:
   → Pressione Ctrl+F
   → Busque por: [opcache]

3️⃣  Adicionar/Modificar estas linhas:

   opcache.enable=1
   opcache.jit_buffer_size=128M
   opcache.jit=1255

4️⃣  Salvar e Fechar (Ctrl+S)

5️⃣  Reiniciar Laragon:
   → Stop All
   → Aguardar 3 segundos
   → Start All

6️⃣  Verificar:
   → http://erpdembena.test/maintenance/settings?activeTab=opcache
   → JIT Status deve mostrar "ON" 🟡


═══════════════════════════════════════════════════════════════════
📌 MÉTODO 2: AUTOMATIZADO (1 minuto)
═══════════════════════════════════════════════════════════════════

1️⃣  Abrir PowerShell como Administrador:
   → Botão direito no menu Iniciar
   → "Windows PowerShell (Admin)"

2️⃣  Navegar até a pasta do projeto:
   cd C:\laragon\www\ERPDEMBENA

3️⃣  Executar o script:
   .\enable-jit.ps1

4️⃣  Seguir as instruções na tela

5️⃣  Reiniciar Laragon quando solicitado

═══════════════════════════════════════════════════════════════════
⚠️  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

❌ JIT ainda mostra "OFF":
   → Verifique se salvou o php.ini correto
   → Reinicie o Laragon completamente
   → Limpe cache do navegador (Ctrl+F5)
   → Execute: http://erpdembena.test/clear-opcache.php

❌ "Failed to load opcache":
   → Verifique se existe: extension=opcache
   → Remova o ";" no início da linha se existir

❌ Buffer size é 0:
   → Certifique-se: opcache.jit_buffer_size=128M
   → Sem espaços extras antes do =
