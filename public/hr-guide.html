1. Introdução ao Módulo de RH

O módulo de Recursos Humanos do ERP Dembena foi desenvolvido para centralizar e facilitar todas as atividades relacionadas à gestão de pessoal. Por meio dele, você pode gerenciar funcionários, documentos, licenças, avaliações e gerar relatórios detalhados.

Gestão de Funcionários

Cadastro completo, histórico profissional e dados pessoais.

Documentos

Armazenamento seguro de todos os documentos relacionados aos funcionários.

Licenças e Ausências

Gestão completa de férias, licenças médicas e outras ausências.

Avaliações

Ferramentas para acompanhar o desempenho dos funcionários.

Importante: O módulo de RH é integrado com outros módulos do sistema, como Folha de Pagamento e Controle de Acesso, permitindo uma gestão completa de recursos humanos.

2. Acesso ao Módulo

1

Como acessar o módulo de RH:

  1. Faça login no sistema ERP Dembena com suas credenciais
  2. No menu lateral esquerdo, localize e clique em "RH" ou "Recursos Humanos"
  3. Você será direcionado para a página principal do módulo de RH, que exibe um painel com resumo dos dados principais
Fig. 1: Localização da opção "Recursos Humanos" no menu lateral
Segurança: Certifique-se de que você possui as permissões adequadas para acessar este módulo. Caso contrário, entre em contato com o administrador do sistema.

3. Cadastro e Gestão de Funcionários

1

3.1 Adicionar um novo funcionário:

  1. Na página principal do módulo RH, clique no botão "Adicionar Funcionário"
  2. Preencha o formulário com os dados pessoais do funcionário:
    • Nome completo
    • Data de nascimento
    • Gênero
    • Número de identificação/CPF
    • Endereço completo
    • Informações de contato (telefone, e-mail)
  3. Preencha os dados profissionais:
    • Cargo
    • Departamento
    • Data de admissão
    • Supervisor direto
    • Tipo de contrato
    • Salário
  4. Clique em "Salvar" para adicionar o funcionário ao sistema
Fig. 2: Formulário para cadastro de um novo funcionário
2

3.2 Editar dados de um funcionário:

  1. Na lista de funcionários, localize o funcionário desejado
  2. Clique no ícone de edição (lápis) ao lado do nome do funcionário
  3. Atualize os campos necessários
  4. Clique em "Atualizar" para salvar as alterações
Fig. 3: Lista de funcionários com botões de ação, incluindo o botão de edição

3.3 Desativar ou reativar um funcionário:

  1. Na lista de funcionários, localize o funcionário desejado
  2. Para desativar: Clique no ícone de status e selecione a opção para desativar
  3. Confirme a ação na mensagem que aparecerá
  4. Para reativar: Acesse a aba "Inativos", localize o funcionário e clique no ícone de status para reativar

4. Gerenciamento de Documentos

4.1 Adicionar documentos a um funcionário:

  1. Na lista de funcionários, clique no nome do funcionário desejado para acessar seu perfil
  2. Na página do perfil, acesse a aba "Documentos"
  3. Clique no botão "Adicionar Documento"
  4. No modal que se abre, preencha:
    • Título do documento (ex: Carteira de Trabalho, Certificado, etc.)
    • Selecione o arquivo do seu computador (formatos aceitos: PDF, JPG, PNG)
    • Data de expiração (se aplicável)
    • Observações (campo opcional)
  5. Clique em "Enviar" para adicionar o documento

4.2 Visualizar ou baixar documentos:

  1. Na página do perfil do funcionário, acesse a aba "Documentos"
  2. Na lista de documentos, clique no ícone de visualização para abrir o documento
  3. Para baixar, clique no ícone de download ao lado do documento desejado

4.3 Excluir documentos:

  1. Na lista de documentos, localize o documento a ser excluído
  2. Clique no ícone de exclusão (lixeira)
  3. Confirme a exclusão na mensagem que aparecerá

5. Licenças e Ausências

Fig. 4: Fluxo de solicitação e aprovação de licenças

5.1 Registrar nova licença ou ausência:

  1. Na página principal do módulo RH, acesse a aba "Licenças"
  2. Clique no botão "Registrar Licença"
  3. No formulário, preencha:
    • Selecione o funcionário
    • Tipo de licença (férias, licença médica, licença maternidade, etc.)
    • Data de início
    • Data de término
    • Motivo ou observações
    • Anexos (se necessário, como atestados médicos)
  4. Clique em "Salvar" para registrar a licença

5.2 Aprovar ou rejeitar solicitações de licença:

  1. Na aba "Licenças", acesse a seção "Pendentes"
  2. Revise os detalhes da solicitação clicando no ícone de visualização
  3. Para aprovar: Clique no botão "Aprovar"
  4. Para rejeitar: Clique no botão "Rejeitar" e informe o motivo
  5. A situação da licença será atualizada no sistema

5.3 Consultar histórico de licenças:

  1. Na aba "Licenças", use os filtros disponíveis para pesquisar por:
    • Funcionário
    • Período
    • Tipo de licença
    • Status (aprovada, rejeitada, pendente)
  2. O sistema exibirá todas as licenças que correspondem aos filtros aplicados

6. Avaliações de Desempenho

6.1 Criar um novo ciclo de avaliação:

  1. Na página principal do módulo RH, acesse a aba "Desempenho"
  2. Clique no botão "Novo Ciclo de Avaliação"
  3. Configure o ciclo:
    • Nome do ciclo (ex: "Avaliação Anual 2025")
    • Período de avaliação (data de início e fim)
    • Departamentos e funcionários participantes
    • Critérios de avaliação
  4. Clique em "Criar Ciclo" para iniciar

6.2 Preencher avaliações:

  1. Na aba "Desempenho", selecione o ciclo de avaliação atual
  2. Localize o funcionário a ser avaliado e clique em "Avaliar"
  3. Preencha cada critério com a pontuação apropriada
  4. Adicione comentários e feedback para cada seção
  5. Clique em "Salvar" para registrar a avaliação

6.3 Revisão e aprovação final:

  1. Depois que todas as avaliações forem preenchidas, acesse "Revisão Final"
  2. Revise cada avaliação e faça ajustes se necessário
  3. Após a aprovação final, as avaliações ficam disponíveis para visualização pelos gestores

7. Relatórios de RH

7.1 Gerar relatórios:

  1. Na página principal do módulo RH, acesse a aba "Relatórios"
  2. Selecione o tipo de relatório desejado:
    • Folha de pagamento
    • Relatório de assiduidade
    • Relatório de turnover
    • Relatório de desempenho
    • Relatório de documentos expirados/a expirar
  3. Configure os parâmetros do relatório (período, departamentos, etc.)
  4. Clique em "Gerar Relatório"

7.2 Exportar relatórios:

  1. Após gerar o relatório, clique em uma das opções de exportação:
    • PDF
    • Excel
    • CSV
  2. O arquivo será baixado automaticamente para seu computador

8. Dicas e Boas Práticas

Mantenha os dados atualizados: Revise e atualize regularmente as informações dos funcionários.
Backup de documentos: Embora o sistema armazene os documentos, mantenha cópias de segurança dos arquivos importantes.
Notificações: Configure alertas para datas importantes, como vencimento de documentos ou aniversários.
Segurança: Não compartilhe suas credenciais de acesso e certifique-se de sair do sistema após o uso.
Treinamento: Ofereça treinamento regular para novos usuários do módulo de RH.
Suporte: Em caso de dúvidas ou problemas, contate o suporte técnico através da opção "Suporte" no menu principal.

Este guia foi criado para auxiliar no uso completo do módulo de RH do ERP Dembena.

Para informações mais específicas ou suporte técnico, entre em contato com a equipe de TI ou consulte a documentação completa do sistema.
