1. Introdução

Este manual tem como objetivo orientar os usuários do sistema ERPDEMBENA sobre como realizar o processamento de folha de pagamento de forma eficiente e precisa. Processamento de folha é uma tarefa crítica que requer atenção aos detalhes e compreensão de todos os elementos envolvidos.

A folha de pagamento no sistema ERPDEMBENA foi desenvolvida para facilitar todo o ciclo, desde a definição de períodos de pagamento, cadastro de itens (como salários, bônus e descontos), até o processamento final e geração de relatórios.

Importante:

Antes de iniciar o processamento da folha, certifique-se de que todos os dados de funcionários estão atualizados no sistema, incluindo informações bancárias e de situação contratual.

2. Conceitos Básicos

Antes de iniciar o processamento da folha, é importante compreender alguns conceitos fundamentais:

Período de Pagamento

Intervalo de tempo sobre o qual a folha é calculada, geralmente mensal.

Itens da Folha

Elementos que compõem a remuneração, como salário base, bônus, horas extras e descontos.

Processamento

Cálculo de todos os valores devidos ao funcionário e aplicação de impostos.

Relatórios

Documentos gerados para consulta e comprovação dos pagamentos realizados.

Fluxo de Trabalho

O processo de folha de pagamento segue tipicamente este fluxo:

  1. Configuração do período de pagamento
  2. Cadastro ou atualização de itens de folha
  3. Processamento da folha para cada funcionário
  4. Revisão e aprovação dos valores
  5. Geração de relatórios e comprovantes
  6. Fechamento do período

3. Períodos de Pagamento

Os períodos de pagamento definem o intervalo para o qual a folha será processada. É essencial configurar corretamente estes períodos antes de iniciar o processamento.

3.1 Acessando Períodos de Pagamento

1 No menu lateral, acesse "Human Resources" > "Payroll Periods"

2 Você verá a lista de períodos já cadastrados no sistema

3.2 Criando um Novo Período

1 Clique no botão "New Period" no canto superior direito

2 Preencha os campos do formulário:

  • Name: Nome do período (ex: "Abril 2025")
  • Start Date: Data inicial do período
  • End Date: Data final do período
  • Payment Date: Data prevista para pagamento
  • Status: Status inicial do período (normalmente "Open")
  • Remarks: Observações adicionais (opcional)

3 Clique em "Save" para criar o período

3.3 Gerenciando Períodos

Cada período pode estar em um dos seguintes estados:

  • Open: Período aberto para inclusão de itens e processamento
  • Processing: Folha está sendo processada, validações estão em andamento
  • Closed: Período encerrado após pagamento, não permite mais alterações

Para editar um período, clique no ícone de lápis na coluna de ações.

Atenção:

Um período marcado como "Closed" não pode mais receber novas entradas ou modificações. Certifique-se de que todos os dados estão corretos antes de fechar um período.

4. Itens da Folha

Os itens da folha de pagamento incluem todos os valores que compõem a remuneração dos funcionários, incluindo salários, bônus, horas extras, bem como deduções e impostos.

4.1 Acessando Itens da Folha

1 No menu lateral, acesse "Human Resources" > "Payroll Items"

2 Você verá a lista de itens já cadastrados, com filtros por tipo e período

4.2 Criando um Novo Item

1 Clique no botão "New Item" no canto superior direito

2 Preencha os campos do formulário:

  • Payroll: Selecione o funcionário e período ao qual o item se aplica
  • Type: Tipo do item (listado abaixo)
  • Description: Descrição detalhada do item
  • Amount: Valor do item
  • Is Taxable: Marque se o item deve ser considerado para cálculo de impostos

3 Clique em "Save" para criar o item

4.3 Tipos de Itens

O sistema permite os seguintes tipos de itens:

Tipo Descrição Exemplo
Allowance Subsídios ou abonos adicionais Auxílio transporte, auxílio alimentação
Bonus Pagamentos extras por desempenho Bônus anual, premiação por metas
Overtime Horas extras trabalhadas Horas extras a 50%, horas extras a 100%
Deduction Descontos diversos Adiantamentos, faltas, danos a equipamentos
Tax Impostos sobre o salário IRRF, impostos locais
Social Security Contribuições para seguridade social INSS, previdência complementar

Dica:

Utilize a opção "Is Taxable" apenas para itens que devem ser incluídos no cálculo de impostos. Itens como reembolsos normalmente não são tributáveis.

5. Processamento da Folha

Após configurar os períodos e cadastrar os itens, é hora de processar a folha de pagamento para cada funcionário.

5.1 Acessando o Processamento

1 No menu lateral, acesse "Human Resources" > "Payroll Processing"

2 Selecione o período de pagamento que deseja processar

5.2 Iniciando um Novo Processamento

1 Clique no botão "New Payroll"

2 Selecione o funcionário para o qual deseja processar a folha

3 Escolha o período de pagamento correspondente

4 O sistema carregará automaticamente o salário base do funcionário

5 Adicione outros itens necessários (bônus, horas extras, etc.)

6 O sistema calculará automaticamente os impostos baseados nos itens tributáveis

7 Verifique o valor líquido calculado e clique em "Save"

5.3 Processamento em Lote

Para processar múltiplos funcionários de uma vez:

1 Na tela principal de processamento, clique em "Batch Processing"

2 Selecione o período e o departamento (opcional)

3 Marque os funcionários que deseja incluir no processamento

4 Clique em "Process Selected"

5 O sistema processará todos os funcionários selecionados usando dados padrão

6 Após o processamento em lote, revise individualmente cada folha para ajustes específicos

Importante:

O processamento em lote é uma ferramenta poderosa, mas requer revisão manual posterior para garantir que todos os casos específicos foram tratados corretamente.

6. Aprovação e Fechamento

Após o processamento, a folha deve passar por um processo de verificação, aprovação e fechamento.

6.1 Verificação

1 Acesse a lista de processamentos para o período atual

2 Revise cada folha processada, verificando valores e cálculos

3 Utilize os filtros para identificar casos excepcionais ou valores anormais

4 Para cada folha, clique no ícone de visualização para verificar os detalhes

6.2 Aprovação

Uma vez verificadas, as folhas precisam ser aprovadas:

1 Na tela de processamento, selecione as folhas a serem aprovadas

2 Clique no botão "Approve Selected"

3 Confirme a ação quando solicitado

4 Após aprovação, as folhas não poderão mais ser modificadas sem autorização especial

6.3 Fechamento do Período

Após todas as folhas serem aprovadas e os pagamentos realizados:

1 Acesse "Human Resources" > "Payroll Periods"

2 Localize o período atual e clique no ícone de edição

3 Altere o status para "Closed"

4 Adicione observações relevantes se necessário

5 Clique em "Save" para confirmar o fechamento

Segurança:

Apenas usuários com permissões adequadas podem aprovar folhas e fechar períodos. Isso garante a segregação de funções e aumenta a segurança do processo.

7. Relatórios

O sistema oferece diversos relatórios para auxiliar na gestão da folha de pagamento e atender requisitos legais.

7.1 Relatórios Disponíveis

Os principais relatórios disponíveis são:

  • Resumo da Folha: Visão geral de todos os pagamentos do período
  • Recibos de Pagamento: Comprovantes individuais para cada funcionário
  • Relatório de Encargos: Detalhamento de impostos e contribuições
  • Relatório por Departamento: Resumo agrupado por departamento
  • Histórico de Pagamentos: Evolução dos pagamentos por funcionário

7.2 Gerando Relatórios

1 No menu lateral, acesse "Human Resources" > "Reports"

2 Selecione o tipo de relatório desejado

3 Configure os filtros (período, departamento, funcionário, etc.)

4 Clique em "Generate Report"

5 O relatório será exibido na tela, com opções para exportar para PDF ou Excel

7.3 Recibos de Pagamento

Para gerar recibos individuais:

1 Na tela de processamento da folha, selecione o funcionário

2 Clique no botão "Generate Pay Slip"

3 O recibo será gerado em formato PDF

4 Você pode imprimir ou enviar digitalmente para o funcionário

7.4 Exportação de Dados

Para exportar dados para outros sistemas:

1 Na tela de relatórios, selecione "Data Export"

2 Escolha o formato de saída (CSV, Excel, XML)

3 Configure os filtros e campos a serem incluídos

4 Clique em "Export" para baixar o arquivo

8. Dicas e Melhores Práticas

Para garantir um processamento de folha eficiente e sem erros, siga estas recomendações:

Planejamento

Estabeleça um calendário de processamento de folha com todas as datas importantes, incluindo prazos para entrada de dados, aprovação e pagamento.

Checklist

Crie um checklist para cada ciclo de processamento, garantindo que nenhum passo importante seja esquecido.

Atualizações

Mantenha os dados dos funcionários sempre atualizados, especialmente informações bancárias e alterações salariais.

Backup

Realize backups regulares dos dados de folha de pagamento, especialmente antes e depois de cada processamento.

8.1 Evitando Erros Comuns

  • Verificação Dupla: Sempre verifique os valores e cálculos antes da aprovação final
  • Teste de Validação: Use os relatórios de resumo para identificar discrepâncias
  • Documentação: Documente todas as alterações manuais feitas na folha
  • Treinamento: Garanta que todos os usuários do sistema estejam devidamente treinados

8.2 Segurança e Confidencialidade

Lembre-se que informações de folha de pagamento são altamente confidenciais:

  • Nunca compartilhe senhas de acesso ao sistema
  • Utilize sempre computadores seguros para processar a folha
  • Documente quem aprovou cada folha e quando
  • Limite o acesso aos relatórios apenas aos usuários autorizados
  • Mantenha-se atualizado sobre as melhores práticas de segurança

Dica Final

Estabeleça uma rotina regular de auditoria da folha de pagamento, verificando aleatoriamente alguns processamentos para garantir a precisão e conformidade com as políticas da empresa e a legislação aplicável.
